# 比例消消（Ratio Chain）— Replit Agent 开发提示词

---

## 项目概述

做一个**教室大屏触控、左右分屏的双人对战数学游戏**，知识点是「比与比例式」（等值比、化简比、连比、解未知数），面向国中七年级（12–13 岁）。

玩法是 **Candy Crush / Two Dots 式的拖拽消除**：在一块稳定的数字宝石盘上，玩家用手指**沿横竖方向**划过相邻格，凑出一条**每段比都相等的比例链**（如 `6:9 = 8:12`）把它们消掉；消除后上方宝石带重力下坠、顶部补新格。**必须有 Candy Crush 般弹润、爽脆的消除与掉落手感（juice）**。游戏须能独立运行，不依赖任何外部系统。

---

## 技术要求

- 框架：React（Vite 脚手架）；语言：JavaScript（不用 TypeScript）。
- 单页应用，尽量少文件。棋盘用 CSS Grid 或 Canvas 均可，但**触控拖拽必须流畅**、动画必须顺滑（用 requestAnimationFrame / CSS transition，全部 ease-out）。
- 触屏优化：所有按钮 ≥48px；根元素 `touch-action:none`（防拖拽时页面滚动/缩放）。
- 目标分辨率横屏大屏 **1920×1080**，需响应式缩放，两侧盘面等宽。
- 输入用 **Pointer Events**（触控+鼠标统一）：`pointerdown`（在盘上）、`pointermove`/`pointerup`（在 document）；移动时用 `document.elementFromPoint` 找当前格。
- 两名玩家在同屏各占左右半边，**各自独立处理自己半边的 pointer 事件**（用坐标归属区分左右，支持两指同时拖）。

---

## 游戏界面布局

整屏横向三分：
- **左 42% / 右 42%**：P1 / P2 盘面区，各是 **8 列 × 7 行** 的数字宝石网格，左右镜像。每侧上方显示该玩家分数、最长链；每侧有一个「💡深提示」按钮。
- **中 16%**：共享视觉带——顶部回合倒计时大字、回合数、比分、回合胜负标记。不放可操作元素，双方盘面独立、不抢目标。
- **底部细条**：每侧一行，显示当前拖拽链读作的比 + ✓/✗ + 预计得分。
- **解未知数弹框**：需要时在对应玩家一侧居中弹出。

---

## 核心游戏循环（每位玩家独立并行）

```
① 玩家目视盘面（有常亮「种子对」提示标出一个可用比的开端）
② 手指按住起点宝石，沿【横、竖四方向】划过相邻格形成链（不能斜），松手提交
③ 判定：链按顺序两两读成比 [c0:c1, c2:c3, …]
   要求 (a) 链长偶数且 ≥4；(b) 所有比化简后完全相等且方向一致（2:3 ≠ 3:2）
   · 若链含「?」格 → 弹出【解未知数】弹框（见下）
   成功 → 宝石依次爆消 → 上方下坠 → 顶部补新格（全程 juicy 动画）→ 结算得分
   失败（比不等 / 未知数选错）→ −2 分 + 该链红色抖动
```

盘面始终稳定——只有玩家主动消除时才发生下坠与补格，**没有下落淹没压力**。

---

## 棋盘生成规则

- **加权数字池：** 从 `{1,2,3,4,5,6,8,9,10,12,15,18,20,24,36}` 按权重抽取，权重：
  `{1:5, 2:7, 3:7, 4:8, 5:6, 6:10, 8:6, 9:6, 10:7, 12:9, 15:5, 18:7, 20:5, 24:6, 36:6}`。
- **通配格「?」：** 6% 概率生成（`UNKNOWN_PROB=0.06`），紫底问号。
- **补格：** 消除后每列剩余宝石落到底，空出的顶部用池子随机补新格。**两名玩家各自独立随机**。
- **可解性保障：** 开局与每次补格后，用 DFS 确认盘面至少存在 1 条**全为已知数字**（不含「?」）的有效 4 链；否则重排该盘。
- **相邻/连接：只横竖 4 方向**（上下左右），不允许斜向连接。

---

## 判定与计分

**比例链判定（伪代码）：**
```
把链的数值按顺序两两配对: pairs = [(c0,c1),(c2,c3),...]
链长必须偶数且 ≥4
每个 (a,b) 化简 gcd 后得 a':b'；所有 pair 的 a':b' 必须完全相同（方向敏感）
=> 相等则有效
```

**计分（只有两个因子）：`得分 = 数字分 × 连比倍率`**
- **数字分** = 每段比取较大的那个数，累加。例 `3:4=6:8=9:12` → 4+8+12 = 24。
- **连比倍率** = {2段:1, 3段:2, 4段:3.5, 5段:5, 6段起: 7+(段数−6)×2}。
- **单次得分** = `max(1, round(数字分 × 连比倍率))`。
- **失败** = −2 分（比不等 或 未知数选错）。

**范例：** `3:4=6:8=9:12` → 24×2 = **48**；`18:24=15:20` → 44×1 = **44**；`4:3=8:6`(?=8) → 12×1 = **12**。

**解未知数（「?」）交互：**
- 一条链**最多含 1 个「?」**（超过则提示「一次只能解 1 个未知数」并判无效）。
- 由已知的比确定最简比 `p:q`，再由「?」所在段的已知伙伴算出应填的值（须为正整数；如 `6:9=?:12` → ?=8）。若算不出正整数（如 `5:?=3:4`）判该链无效。
- 松手时若链含「?」，**弹出弹框**：显示 `6:9 = ?:12　? = ?` + **4 个选项按钮**（1 正确 + 3 干扰整数，随机排列）。
- 选**对** → 用该值补入 → 触发消除并计分（解出的数照常参与数字分）；选**错** → −2 分。
- 弹框弹出时该侧**暂停倒计时**，选完恢复。

## 提示系统

- **常亮「种子对」提示（默认开）：** 用发光/橙色虚线圈住盘面上**一对**宝石——它是某条真实有效链的**头两格**（一个比的开端）。随盘面自动刷新。**不经过「?」格。** 可在设置里关闭。
- **「深提示」按钮：** 点一下高亮**一条完整 4 格链**（约 3 秒后淡出）。代价：**该次消除得分减半**（`round(得分×0.5)`）。按钮有 ~2.5 秒冷却。

---

## 难度曲线（按回合）

| 回合 | 数字池 | 通配率 | 特征 |
|------|--------|--------|------|
| 第 1 局 | 偏小数字、多直接相等比 | 0.04 | 热身 |
| 第 2 局 | 引入需化简的比（6:9 与 8:12） | 0.06 | 考约分 |
| 第 3 局 | 更多大数字(18/20/24/36)、更多「?」 | 0.08 | 考连比/化简/解未知数 |

---

## 赛制与时间

- **三局两胜**：每局限时 **80 秒**刷高分，分高者赢该局，先赢两局者胜整场。
- 顶部回合倒计时，最后 10 秒变红闪烁；解未知数弹框期间该侧暂停计时。
- 开局 3-2-1 倒数后双方同时开始。
- 也提供**单人模式**：单局 120 秒刷高分。

---

## Candy Crush 体感（Juice）—— 重点，务必做足

所有过渡 ease-out，杜绝生硬瞬变。

- **拖拽中：** 起点宝石轻微放大+高亮光环；路径一节节延伸的发光连线；每经过一颗宝石轻微 squash 弹一下。
- **消除瞬间：** 链上宝石**依次错开约 40ms** 爆成彩色粒子（粒子色=该数字专属色）；伴随**上行音阶**脆响（链越长音越高越密）；**屏幕轻微震动**（链越长越强）；分数处飘金色「+48」上浮淡出，分数滚动增长。
- **下坠补格：** 上方宝石**重力加速**下落，落地 squash-stretch 回弹；顶部新宝石从盘上方错落掉入。
- **连比/大链：** ≥3 段比弹出「🔥连比 ×N！」横幅 + 金色闪光 + 更强震动 + 更华丽粒子拖尾。
- **解未知数弹框：** 卡片 scale+opacity 弹入；选对时数字「飞入」? 格发光确认再消除；选错卡片红色抖动。
- **其他：** 种子提示柔和呼吸光晕；空闲 ~5 秒种子对轻微跳动提醒；失败链红色抖动+低沉哔。
- **音效（Web Audio 合成）：** 消除叮（音阶随链长）、下落「啵」、大链号角、解对「叮咚」、失败低哔；一键静音，默认开。
- **视觉基调：** 圆润宝石、明快高饱和配色、柔和阴影高光——整体 Candy Crush 甜润风。数字用颜色+字形双重区分（无障碍）。

## 独立运行要求

- 首页开始菜单：选模式（双人三局两胜 / 单人 120 秒）、静音开关、开始按钮；点开始 3-2-1 倒数进入。
- 不依赖任何外部 URL 参数或后端；不使用持久化存储。
- 整场结束显示结算：双方/本人分数、胜方、最长链与链长分布（4/6/8/10+ 各几次）、解未知数正确率、「再玩一场」按钮。

## 验收清单（部署后逐条测试）

1. 首页可选双人/单人并 3-2-1 倒数进入；双方盘面各 8×7 铺满数字宝石。
2. 只能沿**横竖**方向连；斜向拖拽不选中相邻格。
3. 左右两侧可**同时**各用一根手指拖拽，互不干扰。
4. 拖出 `3:4 → 6:8 → 9:12` → 成功消除，得 **48** 分（数字24×连比2），并有粒子+音阶+下坠补格动画。
5. 拖出 `2:3 → 3:2`（方向相反）→ 失败，−2 分，红色抖动。
6. 拖出经过「?」的链（如 `6:9 → ?:12`）→ 松手弹出解未知数框，4 选项含正确的 8；选 8 → 消除计分；选错 → −2。弹框期间该侧倒计时暂停。
7. 一条链含 2 个「?」→ 提示「一次只能解 1 个未知数」并判无效。
8. 常亮橙色「种子对」提示存在且随每次消除刷新，且从不圈住「?」格；设置里可关闭。
9. 点「深提示」→ 蓝色虚线圈出一条 4 格链，约 3 秒淡出，按钮进入冷却；照它消除时该次得分减半。
10. 消除有明显 juice：粒子爆裂、上行音阶、屏幕轻微震动、金色「+分」上浮、宝石重力下坠+落地回弹。
11. ≥3 段连比触发「🔥连比 ×N！」横幅与更强特效。
12. 三局两胜流程正确：每局 80 秒，分高者赢局，先赢两局者胜整场；结算面板数据完整。
13. 触控与鼠标都能玩；拖拽时页面不滚动不缩放；全程无报错。
